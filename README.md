# SQL Interview Prep & Practice Repository

## 📌 Overview
This repository contains a structured collection of SQL problems, solutions, and mini-projects designed to strengthen SQL skills for interviews and real-world data analysis.

It includes problems from platforms like LeetCode, as well as custom datasets and case studies.

---

## 🎯 Objectives
- Improve SQL problem-solving skills
- Prepare for data analyst / data science interviews
- Practice real-world data scenarios
- Build a strong SQL portfolio

---

## 🛠️ Topics Covered
- Basic Queries (SELECT, WHERE, ORDER BY)
- Joins (INNER, LEFT, RIGHT, FULL)
- Aggregations (COUNT, SUM, AVG, GROUP BY)
- Subqueries & Nested Queries
- Window Functions
- Common Table Expressions (CTEs)
- Data Cleaning & Transformation
- Performance Optimization

---
